# Food Delivery Database System

This project represents a complete database system design for a food delivery application.  
It covers the full database development process starting from conceptual design to implementation.

## Project Components

- **ERD (Entity Relationship Diagram)**  
  Represents the conceptual design of the database and the relationships between entities.

- **Mapping Diagram**  
  Shows how the ERD is mapped into relational tables.

- **Database Schema**  
  Defines the structure of the database tables, attributes, primary keys, and foreign keys.

- **SQL Implementation**  
  A demonstration video that includes all SQL queries such as:
  - Table creation
  - Insert operations
  - Select queries
  - Joins
  - Constraints

## Tools Used
- MySQL / SQL
- Database Design Concepts
- ERD Modeling

## Project Objective

The goal of this project is to design a well-structured and efficient database system that can be used as the backend for a food delivery application.
